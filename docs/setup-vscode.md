# *My Competitive Programming Setup*

*Setup* untuk programming, khususnya untuk *competitive programming* seperti tes praktik di OSN-P dan OSN, setiap cenderung berbeda-beda mulai dari OS-nya, *text editor* atau IDE-nya, hingga *compiler*-nya. Berikut adalah *setup* yang saya sendiri pakai:

- *Text editor* [Visual Studio Code](https://code.visualstudio.com)
- *Compiler* g++ dari [MinGW-w64](https://sourceforge.net/projects/mingw-w64/)
